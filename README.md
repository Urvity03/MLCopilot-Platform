# MLCopilot Platform

An end-to-end AI/ML platform for building, training, deploying, and monitoring machine learning models.

## Features

- Machine Learning model management
- Experiment tracking
- Dataset management
- Model deployment
- Performance monitoring
- Authentication & authorization
- REST API
- Docker support

---

## Tech Stack

### Frontend

- Next.js
- TypeScript
- Tailwind CSS

### Backend

- FastAPI
- Python
- PostgreSQL

### DevOps

- Docker
- GitHub Actions

---


---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Urvity03/MLCopilot-Platform.git
cd MLCopilot-Platform
```

Install dependencies:

```bash
npm install
```

Run the frontend:

```bash
npm run dev
```
